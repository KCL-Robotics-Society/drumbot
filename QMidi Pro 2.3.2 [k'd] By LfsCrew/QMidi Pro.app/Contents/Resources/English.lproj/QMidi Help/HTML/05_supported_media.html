Supported Media

Supported File Formats

  • MIDI files format 0 and 1 (.mid .kar).
  • Audio files formats supported by OS X Core Audio. This includes AIFF, Waves, AAC, Apple Lossless, MP3.
  • FLAC files (up to 24bit/96Khz).
  • Video/Audio formats supported by QuickTime. You can extend supported file formats by installing third party "QuickTime Components" like these:
    • Perian (FLV and a lot of others)
    • Flip4Mac (WMV, AVI)

Playing a MIDI file in sync with an Audio/Video file (QMidi Pro only)

  1. Copy the MIDI file to the AV file's directory.
  2. Rename it as the AV file, adding "mid" or "kar" as extension.
  3. Launch the AV file. The MIDI will be played perfectly in sync.

Supported Karaoke Formats

Text

The more common karaoke format. Karaoke lyrics are stored as text inside the media file. QMidi supports many MIDI karaoke formats and also reads MP3 ID3 lyrics (SYLT, Lyrics3).

Movie

You can find a lot of karaoke files in movie format. QMidi can display any movie readable by Mac OS QuickTime.

CDG

QMidi supports playback of "ripped" CDG files: If a CDG file is paired with the audio file ("songname.mp3" <=> "songname.cdg"), CDG graphics are displayed in the karaoke/mirror windows.

NB: QMidi cannot read directly from CD+G compact discs. You should first extract the Audio+CDG files with dedicated softwares.
