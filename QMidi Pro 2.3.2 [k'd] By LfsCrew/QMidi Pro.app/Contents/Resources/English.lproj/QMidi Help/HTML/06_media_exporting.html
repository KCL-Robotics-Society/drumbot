Media Exporting

QMidi allows you to export media to various formats. This operation is done by choosing "Export Currentâ€¦" from the file menu. Modifications on time stretch / pitch transpose are automatically applied to exported files. The available export formats depends on the source media type.

MIDI To Any

Save Mix

The current mix is saved to the destination file. Because the mix is "static", any automation previously present in the file will be lost.

Remove Muted Channels

The MIDI events targeting the currently muted channels will be ignored.

MIDI To MIDI

Merge Text

This option is available only if an external text file is linked to the current song. It allows you to "embed" the synced text in the destination file (as lyrics meta events).

Any To Audio

Exported audio format can be either AIFF, WAVE, AAC o Apple Lossless. MIDI files are rendered using the selected DLS Music Device SoundFont.

Merge Text (MP3 only)

This option is available only if an external text file is linked to the current song. It allows you to "embed" the synced text in the destination file (as SYLT ID3 tag).

NB: MP3 format is enable only if the source file is already in MP3 format and no audio processing (pitch/tempo change) is needed .

AUDIO/MIDI To QuickTime Movie

If there is any visual content (text karaoke, CDG), this will be exported in the video track of the movie. Otherwise the movie will only contain the audio track.

  • Text karaoke can be saved using several resolutions (including the current karaoke window size).
  • CDG graphics are always saved at their native resolution (300 * 216 pixels).

Text Karaoke To LRC and ChordPro

You can here export the karaoke text to the LRC, LRC Enhanced and ChordPro Formats.
