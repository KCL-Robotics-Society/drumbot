Keyboard/Mouse Shortcuts

In addition to the standard menu shortcuts, you may use the following keyboard/mouse shortcuts to access most QMidi functions.

  SHORTCUT/MODIFIER ACTION
QMidi
  Hold Down Shift Key at startup Reset windows positions
 Hold Down Option Key at startup Refresh library references
Player
 + Increase pitch
  - Decrease pitch
  Click on Pitch Symbol Reset pitch
  > Increase tempo
  < Decrease tempo
  Click on Tempo Symbol Reset tempo
  Space Play/Stop
  Enter Play
  0 (keypad) Stop
  1 (keypad) Rewind
     
Library (item)
 Option-Delete Shows deletion options dialog
     
Library (playlist)
 Option-Delete Delete playlist and remove playlist items from library
     
Play Lists
 Arrow Up Select previous item
  Arrow Down Select next item
  Return (double-click) Load selected item (play it if "AutoPlay" is enabled)
  Tab Cycle playlists
  Hold Down Option Key while opening Refresh playlist references
     
Karaoke
 Option-Click a word/line Move playhead to word's time
(Display)
 Double click anywhere Toggle full-screen mode
(Sync)
 Return Sync next Line
Tab Sync current line
(Movie-CDG)
 Hold Shift Key while resizing window Force to native content aspect ratio
