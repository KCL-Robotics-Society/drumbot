PDF

If a PDF with the same name of the audio/MIDI file ("songname.mp3" <=> "songname.pdf") is found, it is displayed in the karaoke/mirror windows. Use the "Karaoke > PDF" menu to customize the display format and access other useful commands. The same commands are available in the karaoke/mirror contextual menus.

Synchronizing pages

If the PDF has only one page and is displayed as "Single Page", QMidi will automatically scroll the page during playback. If the document contains two or more pages, you will need to sync them to enable automatic page change/scroll.

To synchronize a multipage PDF document:

  1. Choose "Synchronize" from the "Karaoke" menu (karaoke window must be frontmost). This action opens the sidebar with the pages thumbnails.
  2. Start playback. You can also start directly from the location of the song you want to synchronize.
  3. In the sidebar, with the "smiley" cursor , click the page you want to synchronize at the desired time.
  4. Select "Display" from the "Karaoke"menu to exit the synchronization mode.
  5. Choose "Save" from the "File" menu to save your changes.

You can of course do step 3 while the player is stopped. It is not necessary to synchronize all the pages at once: you can adjust the position of a page at any time by rewinding and clicking it at the right time. All actions can be undone using the"Edit > Undo" menu.

Tip: Option-Click a page to move the playhead at the corresponding time.
