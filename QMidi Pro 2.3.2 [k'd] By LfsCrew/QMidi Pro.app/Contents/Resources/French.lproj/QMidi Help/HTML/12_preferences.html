Preferences

Général

Ouvrir fichiers au démarrage

Si cette option est activée, QMidi affichera la boîte de dialogue "Ouvrir" au démarrage de l'application.

Lecture automatique

Si cette option est activée (par défaut), QMidi jouera immédiatement un morceau lorsque celui-ci est chargé. Il peut être utile de désactiver cette fonction pour effectuer un "preroll", afin de permettre un démarrage plus rapide au moment voulu.

Ouverture automatique fenêtre vidéo

Si cette option est activée (par défaut), QMidi affichera automatiquement la fenêtre karaoké à la lecture de vidéo/CDG.

Activer l'historique de lecture

Cette option permet l'enregistrement d'un historique de lecture dans le fichier log de QMidi. Chaque morceau lu génère un rapport au format <date> <heure> <titre>. Cela peut être utile pour savoir quel morceau a été joué durant un spectacle. Le fichier log est situé dans le dossier (Départ)/Bibliothèque/Logs et peut être consulté (normalement avec l'application Console) en choisissant "Historique" dans le menu "Fenêtres".

Lecteur

Messages d'initialisation

Déterminent comment l'appareil MIDI est remis à zéro entre les morceaux, afin d'éviter que les réglages d'un fichier précédent affecte la lecture du fichier actuel. Choisissez une ou plusieurs des options suivantes:

  • GM - Sélectionnez cette option si vous utilisez un synthétiseur externe ou virtuel qui est compatible GM, mais pas GS ni XG.
  • GS - Sélectionnez cette option avec des synthétiseurs compatibles Roland GS et pour des appareils XG en mode GS.
  • XG - Sélectionnez cette option avec des appareils Yamaha XG en mode XG.

Vous pouvez également envoyer ces messages manuellement en choisissant "Réinitialiser système" dans le menu "Lecteur".

Envoyer "All Sounds Off" à l'arrêt

Si cette option est activée (par défaut), QMidi enverra le message de contrôle "All Sounds Off" lorsque vous appuyez sur le bouton stop. Si le périphérique MIDI connecté supporte ce type de commande, il fera immédiatement arrêter tous les sons (sustain des cymbales, réverb, etc.) A ne pas confondre avec le message "All Notes Off" qui est toujours envoyé!

Lecture depuis la première note

Si cette option est activée, QMidi sautera tous les silences situés au début d'un fichier MIDI et commencera immédiatement à la première note.

Pause avant de démarrer

Les synthétiseurs ont besoin d'un peu de temps pour traiter les messages d'initialisation. Il peut donc être nécessaire d'attendre un minimum avant de commencer la lecture. Utilisez la valeur la plus courte possible!

Activer le repérage d'évenements (Chase events)

Lorsque cette fonction est activée, QMidi transmet les messages MIDI nécéssaires chaque fois que vous vous déplacez à une autre position dans votre morceau, afin que tous les appareils MIDI soient correctement réglés au niveau des Program Change, messages de Controller (comme le volume MIDI), etc.

Filtrer les messages redondants

Cette option est activée par défaut et ne devrait pas être désactivée.

Karaoké

Notation des accords

Cette option vous permet de choisir la notation des accords.

Encodage par défaut

Cet encodage sera utilisé par défaut lorsque des fichiers seront ajoutés à une liste de lecture ou bien chargés depuis le Finder. L'encodage utilisé peut être modifié en choisissant "Encodage" dans le menu "Karaoké". Vous pouvez personnaliser la liste des encodages.

Sur texte enrichi, utiliser

Si vous désirez utiliser l'icône smiley sur du texte formaté (RTF) au lieu du texte coloré.

Montrer séparateur texte/accords

Basculer l'affichage du séparateur texte/accords.
