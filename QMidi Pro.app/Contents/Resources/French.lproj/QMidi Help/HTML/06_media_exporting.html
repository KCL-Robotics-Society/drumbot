Exportation de media

QMidi vous permet d'exporter des media en différents formats, en choisissant "Exporter fichier courant..." dans le menu "Ficher". Les modifications de tempo et de hauteur sont automatiquement appliquées aux fichiers exportés. Les formats d'exportation disponibles dépendent du type fichier à exporter.

MIDI vers n'importe quel fichier

Sauvegarder le mix

Le mixage actuel est sauvegardé dans le fichier. Le mixage est "statique"; donc toute automation présente auparavant sera perdue.

Supprimer canaux muets

Les événements MIDI sur des canaux muets seront ignorés.

MIDI vers MIDI

Fusionner texte

Cette option n'est disponible que si du texte externe est synchronisé avec le morceau courant. Elle vous permet "d'inclure" le texte synchronisé dans le fichier de destination (comme des meta événements de paroles).

N'importe quel fichier vers Audio

Le format audio exporté peut être AIFF, WAVE, AAC ou Apple Lossless. Le fichier audio est rendu en utilisant le SoundFont séléctionné dans la fenêtre "Syntétiseur".

Fusionner texte (MP3 seulement)

Cette option n'est disponible que si du texte externe est synchronisé avec le morceau courant. Elle vous permet "d'inclure" le texte synchronisé dans le fichier de destination (comme tag SYLT ID3).

NB: Le format MP3 est disponible uniquement si le fichier source est déjà au format MP3 et aucun traitement audio (hauteur/tempo) n'est nécessaire.

AUDIO/MIDI vers film QuickTime

S'il y a du contenu visuel (texte karaoké, CDG), celui-ci sera exporté sur la piste vidéo du film. Dans le cas contraire, le film ne contiendra que la piste audio.

  • Du texte karaoké peut être sauvegardé en utilisant différentes résolutions (y compris la taille actuelle de la fenêtre karaoké).
  • Les graphiques CDG sont toujours sauvegardés dans leur résolution native (300 x 216 pixels).
