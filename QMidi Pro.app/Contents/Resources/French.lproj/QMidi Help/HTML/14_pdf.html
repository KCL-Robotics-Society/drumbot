PDF

S'il existe un fichier PDF avec le même nom du fichier audio/MIDI ("nomchanson.mp3" <=> "nomchanson.pdf"), il est affiché dans les fenêtres karaoké/miroir . Depuis le menu «Karaoké> PDF", vous pouvez formater l'affichage en fonction de vos préférences et accéder aux différentes commandes pour la navigation. Ces mêmes commandes sont disponibles dans le menu contextuel des fenêtres karaoké/miroir.

Synchronisation des pages

Si le fichier n'a qu'une seule page et le mode d'affichage "Page unique" est utilisé, QMidi fera défiler automatiquement la page au cours de la chanson. Si per contre le fichier contient deux pages ou plus, vous devrez les synchroniser afin de permettre le changement automatique de page et le défilement dans le mode "Page unique".

Pour synchroniser un fichier PDF de plusieurs pages:

  1. Sélectionner «Synchronisation» dans le menu «Karaoké» (avec fenêtre karaoké au premier plan). Cela ouvre le panneau latéral avec les vignette.
  2. Démarrer la lecture. Vous pouvez démarrer directement à partir des points de la chanson que vous souhaitez synchroniser.
  3. Dans la barre latérale, avec le curseur "smiley" actif , cliquer au moment voulu la page a synchroniser .
  4. Sélectionner «Affichage» dans la «Karaoké» pour quitter le mode de synchronisation.
  5. Pour sauvegarder votre travail, choisir "Enregistrer" dans le menu "Fichier".

Vous pouvez bien sûr effectuer l'étape 3 quand le lecteur est arrêté. Il n'est pas nécessaire de synchroniser toutes les pages à la fois: vous pouvez corriger la position d'une page à tout moment en revenant en arrière et en cliquant au bon moment. Toutes ces actions peuvent être annulées par le menu "Edition > Annuler".

Astuce: Option Cliquez sur une page pour déplacer la tête de lecture à la position correspondante.
