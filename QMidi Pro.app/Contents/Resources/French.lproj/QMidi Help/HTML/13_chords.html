Accords

Ajouter/Modifier les Accords

Les accords peuvent être édités en mode "Synchronisation" de la même façon que les paroles (voir "Karaoké"). Soyez sûr de respecter le format indiqué ci-dessous.

Format des Accords

Les accords doivent utiliser la notation standard "A#m7/E", où la première lettre est le nom de l'accord (lettres capitales A-G), la seconde est l'alteration (#, b ou vide), suivie par la description de l'accord. La note de basse peuvent également être ajoutés à la fin, séparée par le caractère "/".

Exemples d'accords:

"B m7"

"Gbsus4"

"D#/F" --> la note de basse est F

"E/F#" --> la note de basse est F#
