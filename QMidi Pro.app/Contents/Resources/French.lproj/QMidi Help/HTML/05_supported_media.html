Formats supportés

Formats de fichiers supportés

  • Fichiers MIDI au format 0 et 1 (.mid .kar).
  • Fichiers audio supportés par Mac OS X Core Audio, y compris AIFF, Wave, AAC, Apple Lossless, MP3.
  • Fichiers FLAC (jusqu'à 24bit/96Khz).
  • Formats Vidéo/Audio supportés par QuickTime. Vous pouvez augmenter le nombre de formats de fichiers supportés en installant des "Composants QuickTime" de tierce partie comme:
    • Perian (FLV et beaucoup d'autres)
    • Flip4Mac (WMV, AVI)

Comment jouer un fichier MIDI synchronisé à un fichier Audio/Vidéo (QMidi Pro seulement)

  1. Copier le fichier MIDI dans le répertoire du fichier Audio/Vidéo..
  2. Donner au fichier MIDI le même nom que le fichier Audio/Vidéo, en ajoutant l'extension ".mid" ou ".kar".
  3. Ouvrir le fichier Audio/Vidéo. Le MIDI sera joué en parfaite synchronisation

Formats karaoké supportés

Texte

Le format de karaoké le plus répandu. Les paroles karaoké sont stockées à l'intérieur du fichier media. QMidi supporte beaucoup de formats MIDI karaoké et lit également les paroles MP3 ID3 (SYLT, Lyrics3).

Vidéo

Beaucoup de karaoké existent sous forme de fichier vidéo. QMidi peut lire n'importe quel fichier lisible par QuickTime.

CDG

QMidi peut lire des fichiers CDG "rippés": Si un fichier CDG est apparié avec un fichier audio ("songname.mp3" <=> "songname.cdg"), les graphiques CDG sont affichés dans la fenêtre karaoké/miroir.

NB: QMidi ne peut pas lire directement des CD+G. Vous devrez d'abord extraire les files Audio+CDG avec un logiciel dédié.
