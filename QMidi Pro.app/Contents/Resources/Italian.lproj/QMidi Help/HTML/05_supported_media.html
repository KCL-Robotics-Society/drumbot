Formati Supportati

Formati File

  • MIDI file formato 0 and 1 (.mid .kar).
  • Formati audio riconosciuti da OS X Core Audio. Tra questi AIFF, Waves, AAC, Apple Lossless, MP3.
  • File audio FLAC (fino a 24bit/96Khz).
  • Formati audio/video riconosciuti da QuickTime. Puoi aggiungere altri formati tramite l'installazione di plugin ("QuickTime Components") come per esempio:
    • Perian (FLV e molti altri)
    • Flip4Mac (WMV, AVI)

Riproduzione di un file MIDI in sincronia con un file Audio/Video (solo QMidi Pro)

  1. Copiare il file MIDI nella stessa cartella del file AV.
  2. Rinominarlo come il nome del file AV e l'estensione "mid" o "kar".
  3. Lanciando il file AV, i due file verranno eseguiti in perfetto sincronismo.

Formati Karaoke

Testo

Il formato karaoke più comune. Il testo è memorizzato all'interno del file multimediale. QMidi supporta molti formati MIDI karaoke e legge anche i testi MP3 (SYLT, Lyrics3).

Filmato

Si possono trovare molti filmati karaoke su internet. QMidi è in grado di visualizzare qualsiasi filmato leggibile da QuickTime.

CDG

Se un file CDG è accoppiato con un file audio ("nomebrano.mp3"<=> "nomebrano.cdg"), la grafica CDG sarà visualizzata nelle finestre karaoke e mirror.

NB: QMidi non può leggere direttamente la grafica CDG dal lettore CD. Dovete prima "estrarre" il contenuto dei CD+G tramite appositi programmi.
