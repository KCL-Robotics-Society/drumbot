Preferenze

Generali

Apri file all'avvio

Se questa opzione è attivata, QMidi presenterà la finestra di dialogo "Apri" in fase di avvio.

Auto play

Se questa opzione è abilitata (default), QMidi inizierà immediatamente la riproduzione del brano caricato. La disattivazione di questa funzione può essere utile per effettuare il "preroll" del brano, permettendo un avvio instantaneo al momento desiderato.

Mostra sempre contenuti video/CDG

Se questa opzione è abilitata (default), QMidi mostrerà automaticamente la finestra di karaoke durante la riproduzione video/CDG.

Registra cronologia dei brani

Questa opzione abilita la registrazione della cronologia di riproduzione nel file log di QMidi. Ogni canzone caricata genera un record con il formato <data> <ora> <titolo>. Questo può essere utile per tenere traccia dei brani suonati durante uno spettacolo. Il file log si trova nella cartella (home)/Libreria/Logs/ e può essere visualizzato scegliendo "Log" dal menu "Finestre" (normalmente tramite l'applicazione Console).

Player

Messaggi di reset

Determinano come il dispositivo MIDI viene inizializzato per evitare che le impostazioni del brano precedente influenzino il brano attualmente in riproduzione. Scegliere una o più delle seguenti opzioni:

  • GM - Selezionare questo messaggio se si utilizza un sintetizzatore GM, ma non compatibile GS/XG.
  • GS - Usare questo messaggio per sintetizzatori Roland GS compatibili XG e dispositivi in modalità GS.
  • XG - Questo messaggio è adatto ai dispositivi Yamaha XG in modalità XG.

È possibile inviare manualmente i messaggi di reset selezionati scegliendo "Resetta dispositivo" dal menu "Player".

Invia "All Sounds Off" allo stop

Se quest'opzione è abilitata (default), QMidi invierà il messaggio di controllo "All Sounds Off" quando si preme il pulsante di stop. Se l'apparato MIDI collegato supporta questo tipo di commando, interromperà immediatamente tutti i suoni (per esempio code dei piatti della batteria, riverberi, etc). Da non confondere con il messaggio "All Notes Off" che viene sempre inviato!

Inizia dalla prima nota

Se quest'opzione è abilitata, QMidi salterà ogni silenzio all'inizio di un file MIDI e inizierà immediatamente la riproduzione con la prima nota.

Aspetta xx millisecondi prima di iniziare

Poiché i sintetizzatori hanno bisogno di tempo per elaborare i messaggi di inizializzazione, può essere necessario dare loro un lasso di tempo prima di cominciare la riproduzione. Utilizzare il minimo valore utile qui!

Abilita funzione chase

La funzione Chase garantisce che gli strumenti MIDI suonino correttamente quando si colloca la barra di trasporto in una nuova posizione e s’avvia la riproduzione. Per farlo, il programma trasmette agli strumenti MIDI una serie di messaggi MIDI ogni volta che nel progetto ci si sposta ad una nuova posizione, assicurando così che tutti i dis- positivi MIDI siano configurati opportunamente per quanto riguarda eventi Program Change, Controller (come il Volume MIDI), ecc.

Salta dati in ridondanza

Questa funzione è abilitata di default e non dovrebbe essere disattivata.

Karaoke

Notazione Accordi

Questa opzione consente di scegliere quale notazione utilizzare per gli accordi.

Codifica testo di default

La codifica specificata qui sarà utilizzata come impostazione predefinita quando si aggiungono file ad una playlist o quando si carica un file dal Finder. Questa potrà comunque essere modificata di seguito scegliendo la voce "Codifica" del menu "Karaoke". È possibile personalizzare l'elenco codifiche.

Su testo formattato usa

Scegliere qui se utilizzare l'icona "smiley" invece del colore su testo formattato (RTF).

Mostra divisore Testo/Accordi

Rende visibile o meno il divisore tra testo e accordi.
