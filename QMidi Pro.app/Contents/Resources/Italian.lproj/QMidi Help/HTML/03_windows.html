Finestre di QMidi

Trasporto (Player)

Questa finestra fluttuante permette di accedere alle principali funzioni del player. E' comunque possibile utilizzare QMidi senza questa finestra, in quanto tutti i comandi, e relative scorciatoie di tastiera, sono accessibili dal menu "Player".

Karaoke/Mirror

Il karaoke (testo o CDG) viene visualizzato in queste finestre. Entrambe le finestre hanno le loro proprie opzioni di formattazione (vedi "Karaoke"). La finestra mirror è normalmente utilizzata a tutto schermo su un secondo monitor.

I filmati vengono visualizzati nella finestra mirror se aperta, altrimenti nelle finestra karaoke.

Libreria

La libreria permette di tenere traccia di tutti i file multimediali utilizzando una singola finestra simile a quella di iTunes. Per caricare i file si usa il pulsante (+) situato in fondo alla lista. È possibile anche trascinare e rilasciare i file dal Finder. Si possono creare playlist e filtri (playlist smart) tramite il pulsante (+) sotto la barra laterale.

NB: Leggi la sezione "Gestione Media" dell'aiuto per imparare il modo più sicuro di archiviare/spostare i file multimediali.

Come impostazione predefinita, facendo doppio clic su un elemento della lista (o premendo il tasto invio mentre è selezionato), il file corrispondente verrà caricato e riprodotto immediatamente. È possibile disabilitare questa funzione (Auto Play) nelle preferenze di QMidi.

  • Se gli elementi di una playlist sono ordinati per indice, è possibile riordinarli tramite drag & drop. È inoltre possibile scorrere fino all'indice desiderato digitando il relativo numero con la tastiera.
  • Se gli elementi sono ordinati per nome, si possono digitare le prime lettere del nome per selezionare il primo elemento corrispondente.

Il campo di ricerca situato nella barra degli strumenti consente di effettuare veloci ricerche sui principali campi descrittivi.

Importazione da iTunes

Si possono importare playlist/selezioni da iTunes semplicemente trascinandole sulla barra laterale della libreria.

Documenti Playlist

QMidi supporta ancora il formato di playlist utilizzato nelle versioni precedenti.

Quando si aprono (dal menu "Archivio/Apri") una cartella o più di un file, una playlist viene automaticamente creata. E' possibile creare una playlist anche scegliendo la voce "Nuova Lista" dal menu "Archivio".

Si possono importare i documenti playlist nella libreria semplicemente trascinandoli sulla barra laterale della stessa.

Mixer MIDI

Il mixer permette di visualizzare e modificare diversi parametri del suono in tempo reale. E' possibile cambiare strumento (nomi GM da 1 a 128), volume, bilanciamento stereo (pan) e livello di riverbero/chorus su uno qualsiasi dei 16 canali MIDI. Se il pulsante "READ" è attivo (default), tutti i parametri vengono letti dal file MIDI e inviati al dispositivo di output. I valori di ogni controllo della finestra del mixer sono aggiornati in tempo reale.

Gli strumenti non GM vengono visualizzati nella forma [P x Bank y] dove "x" è il numero di programma e "y" è il numero di Control Change 0, o numero di banco (Bank Select MSB). Il Banco 127 corrisponde al set di strumenti MT-32 di Roland (non supportato dal sintetizzatore QuickTime).

Per modificare e salvare un mix, si può seguire questa procedura:

  1. Ascoltare l'inizio del file MIDI con il tasto "READ" attivato per caricare il mix registrato nel file.
  2. Disattivare il pulsante "READ".
  3. Effettuare le modifiche.
  4. Salvare il mix dal menu "Archivio/Esporta corrente".

NB: Qualsiasi automazione di missaggio sui parametri gestiti dal mixer verrà cancellata con questa operazione!

Effetti GS

Questa parte del mixer consente di visualizzare/modificare gli effetti master GS. Queste impostazioni hanno effetto solo su dispositivi MIDI compatibili GS, ma eventuali modifiche saranno comunque salvate nel mix.

Cerca Testi

Questa finestra permette di ricercare i testi delle canzoni sul server "LyrDB".
